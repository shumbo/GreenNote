GreenNote

An unofficial Grammarly client for iPad

Grammarly, on your iPad

Grammarly is the best service to improve our writing. It's a pity it doesn't work on an iPad, which becomes a great tool to write stuff with a smart keyboard.

That is why GreenNote is here. GreenNote allows you to use Grammarly on your iPad so that you can write perfect sentences.

Help make the app better

This project is open-sourced on GitHub.
Please feel free to create issues or pull requests to make the app better.
